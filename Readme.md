# India Map Python
Generates a mosaic art of India map from an image using Python.
Uses the PIL library to generate the mosaic art and turtle to draw the map.

## Requirements
- Python 3.6
- PIL
- turtle


## Usage
- Clone the repository
- Run the following command in the terminal
```sh
python india.py
```
